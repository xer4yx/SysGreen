# Versioning: SemVer 0.x from one manual source, surfaced in-app; Conventional Commits

SysGreen versions follow **Semantic Versioning** and sit in the **0.x pre-release line** — no stability promise yet; **1.0.0 is reserved for the first code-signed public release** (ADR-0009). The version has a **single source of truth: `<Version>` in `Directory.Build.props`**, which every assembly (App, Helper, Agent) inherits. The App reads its own `AssemblyInformationalVersion` at runtime and shows `vX.Y.Z` in the main window; any build metadata after `+` (e.g. the git commit .NET appends in a repo, or a future automated stamp) is stripped by a pure `AppVersion.Format` helper in Core, so only the semantic part is shown.

Bumps are **manual for now** — a developer edits the one `<Version>` line — governed by **[Conventional Commits](https://www.conventionalcommits.org/) (Angular types)**: `feat:` → **minor**, `fix:`/`perf:` → **patch**, breaking changes held back below 1.0 (a breaking change bumps the minor, not the major, while < 1.0). `docs:`/`chore:`/`ci:`/`test:`/`refactor:` carry no version bump. The **baseline is set to 0.20.0**: a one-time retroactive reset reflecting the 20 merged PRs (#1–#20) that delivered Tiers 1–3. Earlier history predates this convention and isn't strictly conventional, so the baseline is assigned by inspection rather than recomputed commit-by-commit.

Keeping the number **manual but in `Directory.Build.props`** (not hand-stamped per project, not yet tool-computed) is deliberate: a future tool — **MinVer** or **Nerdbank.GitVersioning**, deriving the version from git tags / conventional-commit history at build time — can take over with **no structural change** and no edits to any consumer. Automating the bump is **explicitly out of scope here but unblocked by this decision**.

**Rejected:** hard-coding a version per project (drift between assemblies); adopting automated/computed versioning now (premature tooling weight before the team needs it); staying at 0.1.0 (hides the maturity the version is meant to communicate).
